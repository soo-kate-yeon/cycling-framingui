# NavigationMenu

`NavigationMenu`는 Radix UI 기반의 접근 가능한 드롭다운 메뉴를 지원하는 수평 네비게이션 컴포넌트다.

## Import

```tsx
import {
  NavigationMenu,
  NavigationMenuList,
  NavigationMenuItem,
  NavigationMenuTrigger,
  NavigationMenuContent,
  NavigationMenuLink,
  navigationMenuTriggerStyle,
} from '@framingui/ui';
```

## 기본 사용법

```tsx
import {
  NavigationMenu,
  NavigationMenuList,
  NavigationMenuItem,
  NavigationMenuLink,
  navigationMenuTriggerStyle,
} from '@framingui/ui';

export function SimpleNav() {
  return (
    <NavigationMenu>
      <NavigationMenuList>
        <NavigationMenuItem>
          <NavigationMenuLink href="/" className={navigationMenuTriggerStyle()}>
            홈
          </NavigationMenuLink>
        </NavigationMenuItem>
        <NavigationMenuItem>
          <NavigationMenuLink href="/about" className={navigationMenuTriggerStyle()}>
            소개
          </NavigationMenuLink>
        </NavigationMenuItem>
        <NavigationMenuItem>
          <NavigationMenuLink href="/contact" className={navigationMenuTriggerStyle()}>
            문의
          </NavigationMenuLink>
        </NavigationMenuItem>
      </NavigationMenuList>
    </NavigationMenu>
  );
}
```

## Props API

### NavigationMenu

| Prop | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | 레이아웃 방향 |
| `dir` | `'ltr' \| 'rtl'` | `'ltr'` | 텍스트 방향 |
| `delayDuration` | `number` | `200` | 드롭다운 표시 전 지연 시간 (ms) |
| `skipDelayDuration` | `number` | `300` | 항목 간 이동 시 건너뛸 지연 시간 |
| `className` | `string` | `undefined` | 추가 유틸리티 클래스 |

### NavigationMenuTrigger

| Prop | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `className` | `string` | `undefined` | 추가 유틸리티 클래스 |
| `disabled` | `boolean` | `false` | 비활성화 상태 |
| `children` | `ReactNode` | - | 트리거 콘텐츠 |

### NavigationMenuContent

| Prop | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `className` | `string` | `undefined` | 추가 유틸리티 클래스 |
| `children` | `ReactNode` | - | 드롭다운 콘텐츠 |
| `forceMount` | `boolean` | `false` | 콘텐츠 강제 마운트 |

### NavigationMenuLink

| Prop | 타입 | 기본값 | 설명 |
| --- | --- | --- | --- |
| `href` | `string` | - | 링크 목적지 |
| `active` | `boolean` | `false` | 활성화 상태 |
| `asChild` | `boolean` | `false` | 자식 엘리먼트로 렌더링 |
| `className` | `string` | `undefined` | 추가 유틸리티 클래스 |

## 디자인 토큰

컴포넌트는 FramingUI 디자인 토큰을 사용한다:

```tsx
// Spacing
--spacing-1  // 메뉴 항목 간 간격
--spacing-2  // 수직 패딩
--spacing-4  // 수평 패딩

// Radius
--radius-md  // 트리거와 콘텐츠의 테두리 반경

// Colors
--bg-background          // 기본 배경
--bg-accent              // 호버/포커스 배경
--bg-accent-foreground   // 호버/포커스 텍스트
--bg-popover             // 드롭다운 배경
--text-popover-foreground // 드롭다운 텍스트
--text-muted-foreground  // 보조 텍스트
--border-default         // 테두리와 인디케이터
```

## 사용 팁

- 드롭다운에는 `NavigationMenuTrigger`, 직접 링크에는 `NavigationMenuLink`를 사용한다
- 접근성을 위해 아이콘 전용 네비게이션 항목에는 `aria-label`을 추가한다
- 드롭다운 콘텐츠는 간결하게—메가 메뉴는 최대 2-3열로 제한한다
- 사이드바 네비게이션에는 `orientation="vertical"`을 사용한다
- 모바일 반응성을 고려—햄버거 메뉴에는 Sheet를 사용한다
- 현재 페이지는 `data-active` 속성으로 표시해 시각적 피드백을 제공한다
- 키보드 네비게이션은 기본 내장—Tab, 화살표 키, Enter, Escape 모두 작동한다
- 드롭다운 중첩은 2단계 이하로 유지하는 게 좋다
